vscode-uni-cloud-snippets #

改动日志

插件特性 #

  • uni-cloud 基本能力代码片段
  • 参考 uni-cloud 官方文档
  • 参考 Vue.js 2 风格指南 和 Vue.js 3 风格指南

插件和文档的冲突之处，请以文档为准。

插件源代码在 ModyQyW/uni-helper。欢迎提交 ISSUE 和 PR 改进本插件。

使用 #

安装插件后重启 VSCode 即可。
