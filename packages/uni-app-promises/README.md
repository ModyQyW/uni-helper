## 起步

### 安装依赖

安装依赖。

使用 `npm`：

```shell
npm install uni-app-promises -D
```

使用 `yarn`：

```shell
yarn install uni-app-promises -D
```

使用 `pnpm`：

```shell
pnpm install uni-app-promises -D
```

使用 `cnpm`：

```shell
cnpm install uni-app-promises -D
```
