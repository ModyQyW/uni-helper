# uni-app-types

`uni-app` 组件类型。维护直到官方类型推出。

**类型和文档的冲突之处，请以文档为准。**

类型源代码在 [ModyQyW/uni-helper](https://github.com/ModyQyW/uni-helper)。欢迎提交 ISSUE 和 PR 改进类型。

## 使用

- 安装依赖

```shell
npm i -D uni-app-types
```

- 配置 `tsconfig.json`

```json
{
  "compilerOptions": {
    "types": ["uni-app-types"]
  }
}
```

- 重启编辑器 / IDE。
